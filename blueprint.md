# Blueprint: 오늘 뭐 먹지?

## 1. 프로젝트 개요

- **목표:** 사용자가 서울 내 특정 지역(구)과 음식 종류를 선택하면, 조건에 맞는 음식점을 무작위로 추천해주는 웹 애플리케이션입니다.
- **핵심 기능:** 서울 열린 데이터 광장의 '일반음식점 인허가 정보' API를 활용하여 실시간으로 운영 중인 음식점 데이터를 기반으로 추천합니다.
- **기술 스택:** 프레임워크 없이 순수 HTML, CSS, JavaScript (ES Modules, async/await, fetch)를 사용합니다.

---

## 2. 디자인 및 기능 변천사

### Version 2: Dark Tech (현재)

- **컨셉:** 어두운 배경에 네온 그린 컬러로 포인트를 준 현대적이고 기술적인 느낌의 디자인입니다. (`study1-5yx.pages.dev` 레퍼런스)
- **색상 팔레트:**
  - 배경: `--bg` (#111314), `--bg2` (#1a1d1f)
  - 텍스트: `--ink` (#e0e0e0), `--ink2` (#9e9e9e)
  - 포인트: `--accent` (#00ff99) 및 글로우 효과
- **타이포그래피:** 'Noto Sans KR' 폰트로 통일하여 깔끔하고 가독성 높은 인터페이스를 제공합니다.
- **컴포넌트 스타일:**
  - 그리드 배경 패턴으로 깊이감을 더했습니다.
  - 입력 필드, 버튼, 카드에 날카로운 테두리와 네온 글로우 효과를 적용하여 미래적인 느낌을 강조합니다.
  - 아이콘과 문구를 '탐색' 컨셉에 맞게 변경했습니다. (예: 🚀 맛집 탐색)

### Version 1: Warm & Modern

- **컨셉:** 따뜻한 베이지색 배경과 부드러운 오렌지-레드 포인트 컬러를 사용한 아늑하고 현대적인 디자인입니다.
- **색상 팔레트:**
  - 배경: `--bg` (#faf7f2)
  - 텍스트: `--ink` (#1c1710)
  - 포인트: `--accent` (#c84b2f)
- **타이포그래피:** 제목에는 'Gowun Batang'을, 본문에는 'Noto Sans KR'을 사용하여 감성적이고 정돈된 느낌을 주었습니다.
- **컴포넌트 스타일:**
  - 부드러운 그림자와 둥근 모서리를 사용하여 전체적으로 편안한 인상을 주었습니다.
  - 결과 카드에 입체적인 레이어 효과를 적용했습니다.

### 핵심 기능 (공통)

- **데이터 연동:** `fetch` API를 사용하여 서울시 공공데이터를 비동기적으로 호출합니다.
- **API 키 관리:** JavaScript 파일 내에서 API 키를 변수로 관리합니다.
- **데이터 캐싱:** 최초 호출 시 API 응답을 변수에 저장하여, 이후의 추천 요청 시에는 다시 API를 호출하지 않고 캐시된 데이터를 사용함으로써 응답 속도를 높이고 API 요청 횟수를 줄입니다.
- **필터링:** 사용자가 선택한 '지역(구)'과 '음식 종류'에 맞춰 '영업/정상' 상태인 음식점만 필터링합니다.
- **랜덤 추천:** 필터링된 목록에서 무작위로 하나의 음식점을 선택하여 추천합니다.
- **결과 표시:** 추천된 음식점의 상호명과 주소를 명확하게 표시합니다.
- **다시 추천:** 사용자가 같은 조건으로 다른 음식점을 추천받을 수 있는 기능을 제공하며, 이전에 추천된 곳은 가급적 제외합니다.
- **UI/UX:** 로딩 및 오류 상태에 대한 시각적인 피드백을 제공하여 사용자 경험을 향상시킵니다.

## 4. 현재 작업 계획 (완료)

- **요청:** 모든 HTML 파일의 GTM 스크립트(Script & Noscript) 설치.
- **작업 단계:**
  1.  ✅ **GTM 스크립트 삽입:** 모든 HTML 파일의 `<head>` 시작 부분에 GTM JS 스크립트 삽입 완료.
  2.  ✅ **GTM noscript 삽입:** 모든 HTML 파일의 `<body>` 태그 바로 뒤에 GTM noscript 코드 삽입 완료.
  3.  ✅ **검증:** 각 파일에 스크립트와 noscript 코드가 올바르게 위치해 있음을 확인 완료.
